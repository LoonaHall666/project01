# Weather Diary (Дневник погоды)
**Автор:** Даутов Самат Фиданович

## Описание программы
Графическое приложение для ведения дневника погоды. Позволяет добавлять ежедневные записи с температурой и описанием погоды в таблицу. Данные сохраняются в файл `weather_data.json`, чтобы не потеряться после закрытия программы.

## Возможности программы
*   **Добавление записей:** Ввод даты (формат ГГГГ-ММ-ДД), температуры и описания.
*   **Фильтрация:** Поиск записей по конкретной дате или по температуре (например: «показать все дни теплее +15°C»).
*   **Сохранение:** Автоматическое сохранение данных в файл.
*   **Просмотр:** Все записи отображаются в удобной таблице.

## Как запустить приложение?
1.  Убедитесь, что у вас установлен Python 3.
2.  Скачайте или клонируйте этот репозиторий.
3.  Откройте терминал/командную строку в папке проекта.
4.  Запустите команду: `python weather_diary.py`
